## 它到底是个什么东西

Total Uninstall 是一款 Windows 平台的卸载工具，开发商叫 Gavrila Martau，从 2005 年做到现在，在这条赛道上深耕了将近二十年。一句话概括它的思路：**不是等你卸载时才动手，而是从你装软件那一刻就开始盯着它。**

具体怎么盯呢？它会在你装新软件之前给系统拍个快照，装完再拍一张，然后把两张照片叠在一起对比，看看这个软件到底动了哪些文件、改了哪些注册表项。等你要卸载它的时候，Total Uninstall 就按这份"案底"一条不漏地清理干净。

这种"安装前监控 + 安装后分析"的双重机制，让卸载的彻底程度远超那些只删主程序的普通工具。对系统洁癖来说，这简直是命定的工具。

## 它能干哪些活

光说"卸得干净"有点抽象，我把它的本事摊开讲讲：

- **分析已装软件再卸载**：对于已经装好但没被监控过的软件，它也能扫描分析、生成安装日志，再按日志精准清除。比直接右键删除靠谱多了。
- **监控新软件的安装过程**：在你装新软件前先开启监控，它会记录软件对系统的每一个改动。卸载时按这份记录操作，干净得像从没来过。
- **系统清理**：顺手扫一遍无用文件和注册表残留，释放点空间。
- **开机启动项管理**：管一管那些悄悄自启的软件、服务、计划任务，开机速度能改善不少。
- **软件备份与迁移**：把已安装的软件打包备份，换电脑或重装系统时直接恢复。专业版还支持在不同电脑之间迁移软件，省得重新装一遍再配一遍。
